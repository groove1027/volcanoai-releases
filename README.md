# volcanoai-releases

**Public mirror for [Volcano](https://volcanoai.io) downloads.**

This repository hosts release binaries (.dmg / .exe / .msi / .deb / .AppImage) for the Volcano desktop app — Korean creators' AI content production OS.

## Downloads

Always grab the latest from: **[volcanoai.io](https://volcanoai.io)** or **[Releases](https://github.com/groove1027/volcanoai-releases/releases/latest)**

| Platform | File |
|---|---|
| macOS Apple Silicon | `Volcano_<version>_aarch64.dmg` |
| macOS Intel | `Volcano_<version>_x64.dmg` |
| Windows (NSIS) | `Volcano_<version>_x64-setup.exe` |
| Windows (MSI) | `Volcano_<version>_x64_en-US.msi` |
| Linux (Debian) | `Volcano_<version>_amd64.deb` |
| Linux (AppImage) | `Volcano_<version>_amd64.AppImage` |

## Installation

See **[volcanoai.io/install](https://volcanoai.io/install)** for OS-specific instructions (Gatekeeper / SmartScreen bypass, etc.).

## Source

The Volcano source code is in a separate (currently private) repository. This mirror exists to serve unauthenticated download URLs for end users.

Releases here are mirrored automatically by the build pipeline in the source repo.
